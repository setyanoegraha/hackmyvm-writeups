# 0x19

This write-up walks through how mission 0x19 was completed, moving from user `frida` to `eliza` and secured `eliza`'s flag.

---

As usual:

```bash
frida@venus:~$ cat mission.txt 
################
# MISSION 0x19 #
################

## EN ##
The password of eliza is the only string that is repeated (unsorted) in repeated.txt.

## ES ##
La password de eliza es el unico string que se repite (sin estar ordenado) en repeated.txt.
```

You can solve this mission by using:

```bash
frida@venus:~$ uniq -d repeated.txt
Fg6b6aoksceQqB9
```

Verify it:

```bash
frida@venus:~$ su - eliza
Password:
eliza@venus:~$ whoami ; id
eliza
uid=1020(eliza) gid=1020(eliza) groups=1020(eliza)
```

Now, go get the flag in the home directory.
