# 0x13

This write-up walks through how mission 0x13 was completed, moving from user `elena` to `alice` and secured `alice`'s flag.

---

As always:
```bash
elena@venus:~$ cat mission.txt 
################
# MISSION 0x13 #
################

## EN ##        
The user alice has her password is in an environment variable. 

## ES ##
La password de alice esta en una variable de entorno.
```

You can look the current environment variables of this session using `env` command:

```bash
elena@venus:~$ env
SHELL=/bin/bash
PWD=/pwned/elena
LOGNAME=elena
MOTD_SHOWN=pam
HOME=/pwned/elena
-------------[SNIP]-------------
USER=elena
PASS=[REDACTED]
```

There is variable `PASS` that stored the password for user `alice`.

Verify it:

```bash
elena@venus:~$ su - alice
Password: 
alice@venus:~$ whoami ; id
alice
uid=1014(alice) gid=1014(alice) groups=1014(alice)
```

Go get the flag:

```bash
alice@venus:~$ cat flagz.txt 
[REDACTED]
```

Objective Secured.
