# 0x14 

This write-up walks through how mission 0x14 was completed, moving from user `alice` to `anna` and secured `anna`'s flag.

---

As usual:

```bash
alice@venus:~$ cat mission.txt 
################
# MISSION 0x14 #
################

## EN ##        
The admin has left the password of the user anna as a comment in the file passwd. 

## ES ##
El admin ha dejado la password de anna como comentario en el fichero passwd.
```

Read the passwd file:

```bash
alice@venus:~$ cat /etc/passwd | grep 'anna'
anna:x:1015:1015::/pwned/anna:/bin/bash
alice@venus:~$ cat /etc/passwd | grep 'alice'
alice:x:1014:1014:[REDACTED]:/pwned/alice:/bin/bash
```

Verify it and get the flag:
```bash
alice@venus:~$ su - anna               
Password: 
anna@venus:~$ whoami ; id
anna
uid=1015(anna) gid=1015(anna) groups=1015(anna)
anna@venus:~$ cat flagz.txt 
[REDACTED]
```
