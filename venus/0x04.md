# 0x04

This write-up walks through how mission 0x04 was completed, moving from user `emma` to `mia`.

---

In this mission our objective is:

```bash
emma@venus:~$ cat mission.txt
################
# MISSION 0x04 #
################

## EN ##
User mia has left her password in the file -.
## ES ##
La usuaria mia ha dejado su password en el fichero -.
```
Since the filname is unusual (it uses a symbol), read the file using `./[filename]`:

```bash
emma@venus:~$ cat ./-
[REDACTED]
```

Switch to user `mia` to verify the password works:

```bash
emma@venus:~$ su - mia
Password:
mia@venus:~$ whoami ; id
mia
uid=1005(mia) gid=1005(mia) groups=1005(mia)
```

Successfully logged in as `mia`. now, read the flag:

```bash
mia@venus:~$ cat flagz.txt
[REDACTED]
```

Objective Secured.
