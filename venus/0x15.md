# 0x15

This write-up walks through how mission 0x15 was completed, moving from user `anna` to `natalia` and secured `natalia`'s flag.

---

As always:

```bash
anna@venus:~$ cat mission.txt 
################
# MISSION 0x15 #
################

## EN ##
Maybe sudo can help you to be natalia.

## ES ##
Puede que sudo te ayude para ser natalia.
```

You can login as natalia using `sudo` command and `-u` option with start a new shell.

```bash
anna@venus:~$ sudo -u natalia bash
```

Voila:
```bash
natalia@venus:/pwned/anna$ whoami;id
natalia
uid=1016(natalia) gid=1016(natalia) groups=1016(natalia)
```

Go get the flag in the home directory of `natalia`.
